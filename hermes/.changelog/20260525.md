# Hermes Agent 更新日志 - 2026-05-25

## 仓库更新

### nousresearch/hermes-agent
- **1 commit** - extending-docker-docs (PR #31775)

**🔄 文档重大扩展：Docker 部署指南**

完整的 Docker 部署文档现已大幅扩展，新增内容包括：

1. **Docker 两种使用模式**
   - Running Hermes IN Docker（容器内运行 agent）
   - Docker as a terminal backend（Docker 作为持久化沙箱执行后端）

2. **Dashboard 运行指南**
   - 环境变量配置（HERMES_DASHBOARD、HOST、PORT、TUI）
   - Dashboard 与 gateway 的 PID 命名空间共享要求

3. **多容器部署模式**
   - 推荐每个 profile 使用独立容器（而非内置多 profile 功能）
   - 隔离性、独立生命周期、端口网络分离等优势

4. **持久化卷结构**
   - 完整的数据目录结构说明
   - `.env`、`config.yaml`、`SOUL.md`、`sessions/` 等路径用途

5. **工具安装指南（5 种方式）**
   - npm/Python 工具：`npx`/`uvx`
   - apt/二进制：安装并记住命令
   - 派生镜像：构建持久化自定义镜像
   - Sidecar 容器：复杂工具/多服务栈
   - 上游贡献：issue 或 PR

6. **本地推理服务器连接**
   - Docker Compose 方式（推荐）：同网络 + 容器名作为 hostname
   - Standalone 方式：host.docker.internal（macOS/Windows）或 --network host（Linux）
   - Ollama 连接配置

7. **Dockerfile 实现细节**
   - 基于 debian:13.4
   - 包含 Python、Node.js、Playwright+Chromium、docker-cli、openssh-client
   - Entrypoint 脚本权限降级机制（hermes 用户）

8. **故障排除**
   - 容器立即退出检查
   - 权限错误处理（HERMES_UID/HERMES_GID）
   - Browser 工具共享内存配置（--shm-size=1g）
   - 网络问题排查
