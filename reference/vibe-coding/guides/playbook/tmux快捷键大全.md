# tmux 快捷键速查表

> 前缀键：`Ctrl+b`

## 会话管理

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `d` | 脱离会话 | 将会话挂起到后台 |
| `s` | 列出会话 | 切换不同会话 |
| `$` | 重命名会话 | 为会话设置描述性名称 |

## 窗口操作

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `c` | 新建窗口 | 创建新的编辑器窗口 |
| `&` | 关闭窗口 | 关闭当前窗口 |
| `n` | 下一个窗口 | 切换到下一个窗口 |
| `p` | 上一个窗口 | 切换到上一个窗口 |
| `0-9` | 切换到第N个窗口 | 快速跳转到指定窗口 |
| `,` | 重命名窗口 | 为窗口设置名称 |
| `w` | 列出窗口 | 查看所有窗口列表 |

## 窗格操作

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `%` | 左右分屏 | 水平分割窗格 |
| `"` | 上下分屏 | 垂直分割窗格 |
| `方向键` | 切换窗格 | 在窗格间移动焦点 |
| `x` | 关闭窗格 | 关闭当前窗格 |
| `q` | 显示窗格编号 | 查看所有窗格编号 |
| `z` | 窗格全屏/还原 | 最大化或还原窗格 |
| `Ctrl+方向键` | 调整大小 | 调整窗格尺寸 |
| `{` / `}` | 交换窗格位置 | 交换相邻窗格 |
| `!` | 窗格转为独立窗口 | 将窗格分离为新窗口 |

## 其他操作

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `[` | 进入复制模式 | 选择和复制文本 |
| `]` | 粘贴 | 粘贴复制的内容 |
| `t` | 显示时间 | 查看当前时间 |
| `:` | 命令模式 | 输入 tmux 命令 |
| `?` | 列出快捷键 | 查看所有快捷键帮助 |

## 命令行

| 命令 | 作用 |
|------|------|
| `tmux` | 新建会话 |
| `tmux new -s 名字` | 新建命名会话 |
| `tmux ls` | 列出会话 |
| `tmux attach -t 名字` | 连接会话 |
| `tmux kill-session -t 名字` | 杀掉会话 |
