# LazyVim 快捷键速查表

## 通用操作

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `<Space>` 等1秒 | 显示快捷键菜单 | 忘记快捷键时查看帮助 |
| `<Space>sk` | 搜索所有快捷键 | 查找特定功能对应的快捷键 |
| `u` | 撤销 | 撤销上一步操作 |
| `Ctrl+r` | 重做 | 恢复被撤销的操作 |
| `.` | 重复上次操作 | 快速重复编辑动作 |
| `Esc` | 退出插入模式/取消 | 返回普通模式或取消操作 |

## 文件操作

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `<Space>ff` | 搜索文件 | 快速打开项目中的文件 |
| `<Space>fr` | 最近打开的文件 | 访问最近编辑过的文件 |
| `<Space>fn` | 新建文件 | 创建新文件 |
| `<Space>fs` | 保存文件 | 保存当前文件 |
| `<Space>fS` | 另存为 | 将文件保存到新位置 |
| `<Space>e` | 打开/关闭侧边栏 | 切换文件浏览器 |
| `<Space>E` | 侧边栏定位当前文件 | 在文件树中高亮当前文件 |

## 搜索

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `<Space>sg` | 全局搜索文本 | 在整个项目中搜索关键词 (grep) |
| `<Space>sw` | 搜索光标下的词 | 搜索当前单词 |
| `<Space>sb` | 当前 buffer 搜索 | 在当前文件中搜索 |
| `<Space>ss` | 搜索符号 | 查找函数、变量等符号 |
| `<Space>sS` | 工作区搜索符号 | 在整个工作区搜索符号 |
| `<Space>sh` | 搜索帮助文档 | 查找 Vim/Neovim 帮助 |
| `<Space>sm` | 搜索标记 | 查找文档中的书签/标记 |
| `<Space>sr` | 搜索替换 | 全局查找并替换 |
| `/` | 当前文件搜索 | 增量搜索当前文件 |
| `n` | 下一个搜索结果 | 跳到下一个匹配项 |
| `N` | 上一个搜索结果 | 跳到上一个匹配项 |
| `*` | 搜索光标下的词 | 快速搜索当前单词 |

## Buffer（标签页）

| 键位 | 动作 | 使用场景 |
|------|------|----------|
| `Shift+h` | 上一个 buffer | 切换到前一个缓冲区 |
| `Shift+l` | 下一个 buffer | 切换到后一个缓冲区 |
| `<Space>bb` | 切换到其他 buffer | 从列表选择缓冲区 |
| `<Space>bd` | 关闭当前 buffer | 关闭当前文件 |
| `<Space>bD` | 强制关闭 buffer | 不保存强制关闭 |
| `<Space>bo` | 关闭其他 buffer | 只保留当前缓冲区 |
| `<Space>bp` | 固定 buffer | 防止 buffer 被关闭 |
| `<Space>bl` | 删除左侧 buffer | 关闭当前左边的缓冲区 |
| `<Space>br` | 删除右侧 buffer | 关闭当前右边的缓冲区 |
| `[b` | 上一个 buffer | 切换到上一个缓冲区 |
| `]b` | 下一个 buffer | 切换到下一个缓冲区 |
