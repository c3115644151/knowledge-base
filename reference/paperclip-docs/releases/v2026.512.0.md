# Paperclip v2026.512.0

> 发布日期：2026-05-13

## 主要变更

### [codex] 本地插件开发工作流改进 (#5821)

- 新增 `paperclipai plugin init` CLI 命令，支持本地插件脚手架创建
- 改进本地插件安装处理、插件路由解析、仪表盘能力行为
- 修复插件 SDK worker 入口验证（支持符号链接包布局）
- 新增插件开发指南文档

### Source-Scoped Recovery Actions (#5599)

- 添加 source-scoped 恢复操作支持

### UI 修复

- 修复 agent 发布评论时的消息归属问题（用户作者ID）

## 技术细节

### CLI 插件初始化
```bash
paperclipai plugin init <plugin-name>
paperclipai plugin install ./my-plugin
```
