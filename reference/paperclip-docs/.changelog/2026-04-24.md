## 核心变更

### SSH 环境支持 (新功能)
paperclip 新增完整的 SSH 环境执行支持，让 AI agent 能够在远程机器上运行工作。

**涉及范围**：
- 共享 adapter 运行时新增 SSH 后端执行目标支持
- 支持远程工作区准备、skill/运行时资源同步、远程会话处理
- 新增多个 adapter 的 SSH 执行覆盖（Claude, Codex, Cursor, Gemini, OpenCode, Pi）
- 新增 `enableEnvironments` 实验性功能标志
- 环境管理 UI 已添加到公司设置、agent 配置、项目配置界面
