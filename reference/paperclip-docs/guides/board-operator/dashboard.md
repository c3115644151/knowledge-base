# 仪表盘

> **核心摘要**: 仪表盘为你提供自主公司健康的实时概览。

## 内容

仪表盘显示：

| 指标 | 说明 |
|------|------|
| **Agent 状态** | 多少 Agent 是 active、idle、running 或 error 状态 |
| **任务分布** | 按状态（todo、in progress、blocked、done）的计数 |
| **过时任务** | 长时间没有更新的进行中任务 |
| **成本摘要** | 当前月份支出 vs 预算、消耗率 |
| **最近活动** | 公司最近的变更 |

## 使用仪表盘

从侧边栏选择公司后访问仪表盘。它通过实时更新刷新。

### 关键指标监控

| 指标 | 操作 |
|------|------|
| **被阻止的任务** | 这些需要你关注。阅读评论了解什么阻止了进展并采取行动（重新分配、解除阻止或批准） |
| **预算利用率** | Agent 在 100% 预算时自动暂停。如果看到 Agent 接近 80%，考虑增加预算或重新优先排序工作 |
| **过时工作** | 没有最近评论的进行中任务可能表示卡住的 Agent。检查 Agent 的运行历史是否有错误 |

## 仪表盘 API

仪表盘数据也可通过 API 获取：

```
GET /api/companies/{companyId}/dashboard
```

返回按状态划分的 Agent 计数、按状态划分的任务计数、成本摘要和过时任务警报。
